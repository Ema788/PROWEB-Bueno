ATRIBUTOS EN HTML

¿Qué son?

Los atributos HTML proporcionan información adicional sobre los elementos HTML.

  1. Todos los elementos HTML pueden tener atributos
  2. Los atributos proporcionan información adicional sobre los elementos
  3. Los atributos siempre se especifican en la etiqueta de inicio
  4. Los atributos generalmente vienen en pares nombre / valor como: name = "value"



Lista de Atributos

Atributo Etiqueta HTML Descripcion
accept form, input Tipos de archivos aceptados por el servidor. Puedes definirlos con valores separados por comas, por ejemplo: .doc, .ods, .txt, etc.
action form El atributo especifica la URL dónde se enviarán los datos recogidos en el formulario, a menudo un archivo .php dentro del mismo servidor.
aling img, table ,tbody ,td ,tfoot ,th ,thead Especifica el alineamiento horizontal del elemento.
async script Indica que el script debe ejecutarse de forma asíncrona.
border img,object ,table Permite definir un borde al elemento HTML./td>
Class Atributo Global Otorga una clase creada en CSS a un elemento HTML. Se usa principalmente para aplicar estilos a elementos con propiedades en común.
color basefont, font
 Atributo usado para indicar el color del texto de un elemento.
controls audio, video Indica si el navegador debe mostrar los controles multimedia del reproductor integrado al usuario.
data object Especifica la URL, absoluta o relativa del objeto o recurso.
disabled button,command ,fieldset ,input Permite que un elemento HTML esté deshabilitado y el usuario no pueda interactuar con el.
enctype form Define el tipo de contenido del formulario cuando el método del mismo es POST.
href a,area ,base ,link La URL de un recurso asociado.
id Atributo Global Usado para asignar un identificador único a un elemento HTML. Usado generalmente para acceder al mismo desde CSS o Javascript. No pueden existir dos «ids» iguales en una misma página web.
name button ,form ,fieldset ,iframe ,input Nombre del elemento. Utilizado a menudo por el servidor para identificar los campos en el envío de formularios.
src audio, embed, iframe, img, input La URL del contenido integrable en la etiqueta HTML que lo acompaña.
