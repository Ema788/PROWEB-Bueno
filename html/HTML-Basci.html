INTRODUCCIÓN A HTML

¿Qué es un elemento HTML?

HTML no es un lenguaje de programación; es un lenguaje de marcado que define la estructura de tu contenido. HTML consiste en una serie de elementos que usarás para encerrar diferentes partes del contenido para que se vean o comporten de una determinada manera.

Un elemento HTML se define mediante una etiqueta de inicio, algo de contenido y una etiqueta de final:


Documentos HTML

Todos los documentos HTML deben comenzar con una declaración de tipo de documento: <!DOCTYPE html>
El documento HTML en sí comienza con <html> y termina con </html>.
La parte visible del documento HTML está entre <body> y /body>.

Ejemplo HTML:

<!DOCTYPE html>
<html>
<title>HTML Tutorial</title>
<body>

<h1>This is a heading</h1>
<p>This is a paragraph.</p>

</body>
</html>


Declaración < !DOCTYPE >

Los <!DOCTYPE> la declaración representa el tipo de documento y ayuda a los navegadores a mostrar las páginas web correctamente.
Solo debe aparecer una vez, en la parte superior de la página ( antes de cualquier etiqueta HTML ).
La declaración no es sensible a mayúsculas y minúsculas y se declara de la siguiente manera:

<!DOCTYPE html>


Encabezados HTML

Los encabezados HTML se definen con el <h1> a <6> etiquetas.
<h1> define el encabezado más importante. <h6> define lo menos importante rumbo:

<h1> Este es el encabezado 1 </h1>
<h2> Este es el encabezado 2 </h2>
<h3> Este es el encabezado 3 </h3>


Párrafos HTML

Los párrafos HTML se definen con la etiqueta <p>

<p> Este es un párrafo. </p>
<p> Este es otro párrafo. </p>


Enlaces HTML

Los enlaces HTML se definen con el <a> etiqueta. Es una etiqueta que nos ayuda a poder crear un enlace a una página web. El atributo principal de la etiqueta HTML es href.
Los href atributo especifica la URL de la página

<a href="https://campusvirtual.uptlax.edu.mx/"> Link al campus virtual UPTx. </a>


Imágenes HTML

Usamos la etiqueta IMG para mostrar imágenes dentro del contenido. Necesita el atributo src para funcionar, ya que será donde indicaremos desde donde tiene que mostrar la imagen.
Las imágenes HTML se definen con el <img> etiqueta.
El archivo fuente (src), texto alternativo (alt), (width), y (height) se proporcionan como atributos:

<img src="UPTx.jpg" alt="campusvirtual.uptlax.edu.mx" width="104" height="142">
