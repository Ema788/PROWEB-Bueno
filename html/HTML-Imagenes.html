¿Cómo ponemos una imagen en una página web?

Para poner una imagen simple en una página web, utilizamos el elemento <img> y img>.

Se trata de un elemento vacío (lo que significa que no contiene texto o etiqueta de cierre) que requiere de por lo menos de un atributo para ser utilizado: src (a veces denominado por su nombre completo, source).

El atributo src contiene una ruta que apunta a la imagen que quieres poner en la página, que puede ser una URL relativa o absoluta, de la misma forma que el elemento <a> y /a> contiene los valores del atributo href

HTML-imagenes
Ejemplos

< img src="nombreimagen.jpg" />

Dimensiones de la imagen: width y height

Si bien disponemos de los atributos width para el ancho de la imagen y height para el alto de la imagen

Titulo de imagenes

Se pueden añadir atributos title a las imágenes para proporcionar más información de ayuda si es necesario

title="Exposición de un T-Rex en el museo de la Universidad de Manchester.">

Texto alternativo

Alt

Su valor debe ser una descripción textual de la imagen para usarla en situaciones en que la imagen no puede ser vista/mostrada o tarde demasiado en mostrarse por una conexión lenta a internet

Imagen de fondo css

La propiedad CSS background-image y las demás propiedades background-* se usan para controlar la colocación de la imagen de fondo.

background-image: url("images/dinosaur.jpg");

GIF

Es un formato para imágenes de mapas de bits las cuales soportan 8 bits por pixel. El formato GIF soporta imágenes animadas.

PNG

Es un formato de imagen en mapa de bits que emplea compresión de datos sin pérdida de información. No requieren de licencia de patente. Es un formato creado para utilizar imágenes en Internet con un tamaño adecuado.

JPEG

son imágenes digitales comprimidas con pérdida de información. Pero que nos permiten tener imágenes digitales que ocupen poco espacio.
