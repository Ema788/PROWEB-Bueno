ATRIBUTOS EN HTML

¿Qué son?

Los atributos HTML proporcionan información adicional sobre los elementos HTML.

  1. Todos los elementos HTML pueden tener atributos
  2. Los atributos proporcionan información adicional sobre los elementos
  3. Los atributos siempre se especifican en la etiqueta de inicio
  4. Los atributos generalmente vienen en pares nombre / valor como: name = "value"



Lista de Atributos

y etiquetas donde aplican

href

La etiqueta define un hipervínculo. El atributo especifica la dirección URL de la página El enlace va a:< a>href

Ejemplo< a href="https://www.paginaejemplo.es" title="Más información" >

src

La etiqueta se utiliza para incrustar un imagen en una página HTML.Especifica la ruta a la imagen que se va a mostrar:< img > src

Ejemplo< img src="https://www.ionos.mx/digitalguidecocherojo.jpg" alt="el coche rojo" >

accept < form>, < input>

Tipos de archivos aceptados por el servidor. Puedes definirlos con valores separados por comas, por ejemplo: .doc, .ods, .txt, etc. O también puedes indicarle un conjunto de tipos de archivos, como por ejemplo: «video/*» o «audio/*»

action < form>, < input>

El atributo especifica la URL dónde se enviarán los datos recogidos en el formulario, a menudo un archivo .php dentro del mismo servidor

aling < img>, < table>, < tbody>, < td>, < tfoot> , < th>, < thead>, < tr>

Especifica el alineamiento horizontal del elemento. Este atributo sigue funcionando pero no es aconsejable usarlo ya que está en desuso. Lo recomendable es usar reglas CSS en su lugar

async < scrip>, < input>

Indica que el script debe ejecutarse de forma asíncrona.

border < img>, < object>, < table>

Permite definir un borde al elemento HTML.

charset < meta>, script

Atributo necesario par definir la codificación de caracteres de la página o del script

class Atributo Global

Otorga una clase creada en CSS a un elemento HTML. Se usa principalmente para aplicar estilos a elementos con propiedades en común.

Ejemplo< class="coches" >

color < basefont>, < font>, < hr>

Atributo usado para indicar el color del texto de un elemento.

Style

Define el estilo de un elemento HTML y puede determinar, por ejemplo, su color, fuente o tamaño

Ejemplo style="color: blue; font-size: 2em"

colspan < td>, < th>

Dentro de una tabla, define el número de columnas que una celda debe ocupar.

content < meta>

Un atributo que permite definir el contenido de la etiqueta HTML < eta> que acompaña.

controls < audio>, < video>

Indica si el navegador debe mostrar los controles multimedia del reproductor integrado al usuario.

Ejemplo< video controls poster="previsualización.png">

data < object>

Especifica la URL, absoluta o relativa del objeto o recurso.

Puedes definirlos con setAttribute y leerlos con getAttribute.

dir Atributo Global

Con este atributo puedes definir la dirección del texto en un elemento HTML. Los valores permitidos son ltr (Izquierda-a-Derecha) o rtl (Derecha-a-Izquierda).

disabled < button>, < command>, < fieldset>, < input>, < keygen>, < optgroup>, < option> , < select>, < textarea>

Permite que un elemento HTML esté deshabilitado y el usuario no pueda interactuar con el.

enctype < form>

Define el tipo de contenido del formulario cuando el método del mismo es POST.

for < label>, < output>

Atributo usado para asociar o describir elementos que pertenecen a éste.

form < button>, < fieldset>, < input>, < keygen>, < label>, < object>, < output>, < select>, < textarea>

Atributo para indicar el formulario al que pertenece el elemento.

id Atributo Global

Usado para asignar un identificador único a un elemento HTML. Usado generalmente para acceder al mismo desde CSS o Javascript

method < form>

Define el método HTTP a emplear para enviar el formulario. Puede ser GET (predeterminado) o POST.

name < button>, < form>, < fieldset>, < iframe>, < input>

Nombre del elemento. Utilizado a menudo por el servidor para identificar los campos en el envío de formularios

placeholder < input>, < textarea>

Proporciona una sugerencia al usuario de lo que puede ser introducido en el campo de un formulario.

required < input>, < select>, < textarea>

Con este atributo hacemos que la etiqueta Html sea, o no, de obligada cumplimentación.

selected < option>

Define un valor que aparecerá por defecto seleccionado al cargar la página.

size < input>, < select>

Define el ancho del elemento (en píxeles).

type < button>, < input>, < command>, < embed>, < object>, script , < source>, style , < menu>

Define el tipo de elemento HTML.

value < button>, < option>, < input>, < li>, < meter>, < progress>, < param>

Define el valor predeterminado a ser mostrado al cargar la página.
